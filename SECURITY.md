## 🇯🇵 日本語

### 脆弱性の報告について

本リポジトリでは GitHub の「セキュリティ勧告」機能を使って、  
**非公開での脆弱性報告**を受け付けています。

[こちらのフォームからご連絡ください](../../security/advisories/new)。  
GitHub のアカウントがあれば誰でも利用できます。

**※ GitHub Issues では報告しないようにお願いします。**

---

## 🇬🇧 English

### Reporting a Vulnerability

This repository accepts **private vulnerability reports** via GitHub's built-in security advisory feature.

Please [submit your report here](../../security/advisories/new).  
Anyone with a GitHub account can use it.

**Do not use GitHub Issues for security-related reports.**

## 🇷🇺 Русский

### Сообщение об уязвимости

Этот репозиторий использует функцию «Security Advisories» на GitHub для приёма  
**конфиденциальных сообщений об уязвимостях**.

Пожалуйста, [отправьте ваш отчёт здесь](../../security/advisories/new).  
Эта функция доступна всем пользователям с аккаунтом GitHub.

**Пожалуйста, не используйте Issues для таких сообщений.**
